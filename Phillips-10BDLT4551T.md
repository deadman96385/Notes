## ADB/Loader Mode
Pop cover off the top and then use the one micro usb that is by itself as the main data port. The other 2 micro usb ports are usb host and not sure on the "expansion port"

## Special Key combos

```Menu + Volume-```   Launch Admin mode app (Password is 1234)

```Menu + Volume+```   Launch Quick Information app

```Menu + Menu + Volume+ + Volume- + Volume- + Volume+```   Enable touch function

```Menu + Menu + Volume- + Volume-```   Turn on the screen
